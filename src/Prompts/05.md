## Prompt para criação de Modal de Adicionar ao Carrinho

Você é um desenvolvedor Full Stack Junior, especializado em React

## Contexto (QUAIS SÃO AS TECNOLOGIAS USADAS)

Você está trabalhando em um projeto de vários restaurantes, que ao clicar em algum deles, abrirá um lista de produtos do restaurante escolhido.

- npm
- React
- Styled componentes
- Redux toolkit

## Regras (USAR LINGUAGENS/TERMOS DETERMINÍSTICOS)

- NUNCA use cores hard-coded, apenas cores do tema que estão em src\styles.ts
- SEMPRE use o MCP do Context7 para buscar documentações, sites e APIs.
- SEMPRE use como refêrencia o que já foi criado.
- SEMPRE mantenha o mesmo padrão e use como referência @src\components para que o código fique limpo e organizado

## Tarefa

- Crie a página do carrinho dentro @src\components conforme esta no figma: https://www.figma.com/design/JjduV2Tg713TzYUUsees8b/efood?node-id=37-125&t=8O2oOeUE0SJH0LVY-4 MCP **SEJA 100% FIEL AO FIGMA, CUSTE O QUE CUSTAR**

## Requisitos funcionais

- Configure o redux toolkit em @src\store criando outras pastas quando necessário para que fique mais limpo e organizado.
- Faça a dinâmica do preenchimento do carrinho através do Redux;
- No carrinho o valor da compra deverá ser a soma do preço dos produtos;
- Crie uma const Store e um type RootReducer com um Typeof dentro de @src\store\index.ts
- Crie uma nova pasta chamada reducers dentro de @src\store e dentro deve conter um arquivo que contem o Slice.
- Utilize o provider em @src\App.tsx com uma store
